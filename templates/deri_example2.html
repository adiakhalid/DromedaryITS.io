Product and Trigonometric Rules

    Example 1: \(y = sin(x)\cdot cos(x)\)

    Solution:

    \(\;\;\;\;\;\dfrac{dy}{dx} = \dfrac{d}{dx}(sin(x)cos(x))\)

    \(⇒ \dfrac{dy}{dx} = cos(x)\cdot\dfrac{d}{dx}(sin(x)) + sin(x)\cdot\dfrac{d}{dx}(cos(x))\)

    \(⇒ \dfrac{dy}{dx} = cos(x)\cdot cos(x) + sin(x)\cdot (-cos(x))\)

    \(⇒ \dfrac{dy}{dx} = cos^2(x) - sin^2(x)\)


    Example 2: \(y = x^2sin^{-1}(x)\)

    Solution:

    \(\;\;\;\;\;\dfrac{dy}{dx} = sin^{-1}(x)\cdot \dfrac{d}{dx}(x^2) + x^2\cdot \dfrac{d}{dx}(sin^{-1}(x))\)

    \(⇒ \dfrac{dy}{dx} = sin^{-1}(x)\cdot 2x+ x^2\cdot \dfrac{1}{\sqrt{1 - x^2}}\)

    \(⇒ \dfrac{dy}{dx} = 2x\cdot sin^{-1}(x)+ \dfrac{x^2}{\sqrt{1 - x^2}}\)


    Example 3: \(y = tan(x)\cdot sec(x) + sin^2(x)\)

    Solution:

    \(\;\;\;\;\;\dfrac{dy}{dx} = (sec(x))\cdot \dfrac{d}{dx}tan(x) + tan(x)\cdot \dfrac{d}{dx}(sec(x))\)
    \(\;\;\;\;\;\;\;\;\;\;\;\; + \dfrac{d}{dx}(sin^2(x))\)
    \(⇒ \dfrac{dy}{dx} = sec(x)\cdot sec^2(x) + tan(x)\cdot (sec(x))(tan(x))\)
    \(\;\;\;\;\;\;\;\;\;\;\;\; + 2sin(x)\cdot cos(x)\)
    \(⇒ \dfrac{dy}{dx} = sec^3(x) + tan^2(x)\cdot sec(x) + 2sin(x)\cdot cos(x)\)

    \(⇒ \dfrac{dy}{dx} = sec(x)(sec^2(x) + tan^2(x)) + 2sin(x)\cdot cos(x)\)
