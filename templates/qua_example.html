Algebra: Quadratic Equation Exmaples

    Example 1: \(3x^2 + 6x - 9 = 0\)

    Solution:

    Split the exponent of degree one
    \(⇒ 3x^2 + 9x - 3x - 9 \,\;\;\;\;= 0\)
    \(⇒ 3x(x + 3) - 3(x + 3) = 0\)
    \(⇒ (3x - 3)(x + 3) \;\;\;\;\;\;\;\;\,= 0\)
    when \(3x - 3 = 0 \;\;\;\;\, ⇒ x = 1\)
    when \(x + 3 = 0 \;\;\;\;\;\;\, ⇒ x = - 3\)
    The required solutions are:
    \(x = -3, 1\)


    Example 2: \(x^2 - 7x + 8 = 0\)

    Solution:

    Split the exponent of degree one
    \(⇒ x^2 - x - 8x + 8 \;\;\;\;\;\;= 0\)
    \(⇒ x(x - 1) - 8(x - 1) = 0\)
    \(⇒ (x - 8)(x - 1) \,\;\;\;\;\;\;\;\;= 0\)
    when \(x - 8 = 0 \,\;\;\;\;⇒ x = 8\)
    when \(x - 1 = 0 \,\;\;\;\;⇒ x = 1\)
    The required solutions are:
    \(x = 1, 8\)


    Example 3: \(2x^2 + 3x - 5 = 0\)

    Solution:

    Split the exponent of degree one
    \(⇒ 2x^2 - 2x + 5x - 5 \;\;\;\;= 0\)
    \(⇒ 2x(x - 1) + 5(x - 1) = 0\)
    \(⇒ (2x + 5)(x - 1) \;\;\;\;\;\;\;\;\,= 0\)
    when \(2x + 5 = 0 \;⇒ x = - 5/2\)
    when \(x - 1 = 0 \;\;⇒ x = 1\)
    The required solutions are:
    \(x = -5/2, 1\)
